Seletores CSS

Bem-Vindo!

No CSS nós possuimos os seletores por ID

Olá tudo bem?

  • Seletor por Tipo: Busca elementos por uma tag HTML especifica
  • Seletor por ID(#): Busca elementos através do atributo "id"
  • Seletor por Classe(.): Busca elementos através do atributo "class"
  • Seletor Universal(*): Seleciona todos os elementos HTML
  • Seletor de Atributo([atrib]): Seleciona elementos que possuem um atributo específico em nosso documentos HTML e conseguimos também buscar atributos com um valor especifico
